# Habr Top Daily Parser

**Минималистичный парсер для сбора заголовков и просмотров статей из топа Habr**

#### (Сделано для https://solvit.space/projects/habr_parser)
---

## 📌 Описание

Скрипт выполняет парсинг страницы `https://habr.com/ru/articles/top/daily/` и извлекает для каждой статьи:
- Заголовок
- Количество просмотров

Результат выводится в консоль и сохраняется в текстовый файл `title.txt`.

Код построен с учётом обработки ошибок и изменений структуры HTML.

---

## 🧰 Технический стек

| Компонент | Назначение |
|-----------|------------|
| Python 3.8+ | Базовый язык |
| `requests` | Выполнение HTTP-запросов |
| `BeautifulSoup (bs4)` + `lxml` | Парсинг HTML |
| `sys.stdout` (с UTF-8) | Корректный вывод в Windows-консоли |

---

## ⚙️ Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/Abemerik/habr-parser.git
cd habr-parser

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python main.py
```

## 📄 Формат выходных данных

### В консоли

```
Title: Название статьи | Views: 12K
Title: Название статьи | Views: 26K
...
 Парсинг успешно завершен
```

### В файле `title.txt`

Каждая строка содержит одну статью в том же формате:

```
Title: Название статьи | Views: 12K
Title: Название статьи | Views: 26K
```
